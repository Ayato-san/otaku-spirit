# Otaku Spirit

This bot is created for the discord server [Otaku Army](https://discord.gg/KJ3WhuegWm)

### Libs used
> discord.js : 13.6.0

> dotenv : 16.0.0

> glob : 7.2.0

> mongoose : 6.2.3

### Tree structure

```
|
|→ commands : all commands and slash commands
|   ↳ utils
|
|→ events : all events
|   ↳ client : default intents
|   ↳ guild_messages : intents for messages
|
↳ utils : all utilities
   ↳ handlers
```
